Commands and Arguments

Bash will read everything you write in your terminal as a command, it takes the first word you type in as a command and all the following words as arguments to that command. Now we will talk about some commands.

ls

The ls command shows you what files are in your current directory. You could also type ls documents to see what files are in your documents directory for example.

Comment

By putting a # infront of whatever you type it turns that into a comment, comments are ignored by the shell and will not be read as a command.

Touch

Touch changes the Last "Modified" time of a file, if you touch a file that didnt exist prior the touch command will make an empty file with that name for you.

rm

The command rm will remove the file or directory that follows, for example if i said "rm media" it would remove a directory named media.

Echo

The command echo will take what ever follows and repeat it once, for example if you write "echo Hello" the next line in the terminal would say Hello.

IMPORTANT

ALL ARGUMENTS SHOULD BE SEPERATED FROM THE COMMAND OR IT WILL NOT WORK FOR EAMPLE "lsdocuments" WONT WORK BECUASE THERE IS NO SPACE BETWEEN THE COMMAND AND THE ARGUMENT.
